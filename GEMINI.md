# Project Context
This is a monorepo for a WhatsApp Linked Device feature.
- **Tech Stack:** Angular 20, .NET Framework 4.7.2 (OWIN WebSockets), Node.js (Express/Baileys).
- **Architecture:** The .NET backend proxies WebSocket connections between the Angular frontend and a local Node.js microservice.
- **Principles:** Strictly enforce SOLID, DRY, and KISS principles.
- **Documentation:** Use JSDoc for Angular and XML docs for .NET.
